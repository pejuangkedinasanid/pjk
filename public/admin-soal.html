Import Soal via JSON
Format harus berupa array JSON. Setiap soal wajib punya kategori, pertanyaan, dan opsi (objek A–E).

• aspek bersifat opsional, tapi sangat disarankan diisi (dipakai di halaman Analisis Perkembangan). Isi salah satu sesuai kategorinya:
  – TWK: Nasionalisme, Integritas, Bela Negara, Pilar Negara, Bahasa Indonesia
  – TIU: Analogi Verbal, Silogisme, Analitis, Berhitung, Deret Angka, Perbandingan Kuantitatif, Soal Cerita, Analogi Figural, Ketidaksamaan Figural, Serial Figural
  – TKP: Pelayanan Publik, Jejaring Kerja, Sosial Budaya, Teknologi Informasi dan Komunikasi (TIK), Profesionalisme, Anti Radikalisme

• TWK / TIU → wajib isi jawaban_benar (satu huruf A–E). Otomatis: benar = 5 poin, salah = 0.
• TKP → wajib isi poin (objek angka 1–5 untuk tiap huruf A–E), jangan isi jawaban_benar.
• gambar_url dan pembahasan sifatnya opsional. 
[
  {
    "kategori": "TWK",
    "aspek": "Pilar Negara",
    "pertanyaan": "Pancasila disahkan pada tanggal...",
